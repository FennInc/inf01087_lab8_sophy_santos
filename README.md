# inf01087_lab8_sophy_santos

Essa atividade consiste em uma pequena pesquisa sobre o uso de LLMs dentro da computação, como elas podem ser úteis e quais riscos podem causar.

Este trabalho utilizou uma ferramenta de IA generativa para auxiliar na geração de explicações e  exemplos de código. 
Todos os prompts utilizados estão documentados em prompts.txt. 
As respostas foram revisadas, testadas e verificadas pelo autor antes de serem incluídas no 
relatório.
